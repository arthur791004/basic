# Git

## commands
* git init
* git config [--global]
  * user.name
  * user.mail
  * alias
* git add <filename>
* git commit
* git status
  * -uno
* git branch
  * git checkout -b <branch>
  * git checkout <branch>
* git merge
* git stash save -u <comment>
* git reset
  * --hard: 捨棄所有還沒 commit 的修改，強制回到上一個版本
* git log
  * --oneline
  * --decorate
  * --graph
  * -<num>
  * -p: 顯示差異
  * -stat: 檢視每個更新的簡略統計資訊
  * git log <new>...<old>
  * git log --after=<time>
  * git log <time> --before
* git diff
  * -p: 顯示差異
* git blame
* git tag
* git rebase
* git pull
* git push
* git fork
* git clone
* git remote add <name> <remote_url>
